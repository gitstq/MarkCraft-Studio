# Contributing to MarkCraft Studio

感谢你对 MarkCraft Studio 的关注！我们欢迎所有形式的贡献。

## 🛠️ 开发环境设置

### 1. 克隆仓库

```bash
git clone https://github.com/gitstq/MarkCraft-Studio.git
cd MarkCraft-Studio
```

### 2. 安装开发依赖

```bash
pip install -e ".[dev,all]"
```

### 3. 运行测试

```bash
pytest tests/ -v
```

## 📝 提交规范

我们使用 [Conventional Commits](https://www.conventionalcommits.org/) 规范：

- `feat: 新增功能`
- `fix: 修复问题`
- `docs: 文档更新`
- `refactor: 代码重构`
- `test: 测试相关`
- `chore: 构建/工具相关`

## 🐛 报告问题

请通过 [GitHub Issues](https://github.com/gitstq/MarkCraft-Studio/issues) 提交问题，包含：

1. 问题描述
2. 复现步骤
3. 期望行为
4. 实际行为
5. 环境信息（Python版本、操作系统等）

## 📄 开源协议

通过贡献代码，你同意你的贡献将在 MIT 协议下发布。
